# Banking System

A professional Banking System built using Python, Tkinter, and SQLite.

This project allows users to register accounts, log in securely, deposit and withdraw money, and monitor transaction history through a graphical user interface.

---

## Features

- User Registration
- Login Authentication
- Deposit System
- Withdraw System
- Balance Monitoring
- Transaction History
- SQLite Database Integration
- Simple and Clean GUI

---

## Technologies Used

- Python
- Tkinter
- SQLite3
- VS Code
- GitHub
